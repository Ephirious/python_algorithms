# Лабораторная работа: Алгоритмический мини-пакет (структуры данных и сортировки)

Этот проект представляет собой набор реализаций ключевых структур данных и популярных алгоритмов сортировки, дополненный инструментарием для их тестирования и анализа производительности. Это идеальная основа для лабораторной работы по дисциплинам, связанным с алгоритмизацией и программированием.

## Структура проекта

Проект организован в логические модули Python, каждый из которых отвечает за определенный аспект лабораторной работы:

*   `stack.py`: **Реализация структуры данных "Стек" (Stack)**. Содержит методы `push()`, `pop()`, `peek()`, `is_empty()`.
*   `abstract_function.py`: **Абстрактный базовый класс** для наследования другими функциями. Определяет общий интерфейс.
*   `factorial.py`: **Реализация функции вычисления факториала** (рекурсивная или итеративная).
*   `fibonacci.py`: **Реализация функции вычисления чисел Фибоначчи**.
*   `benchmarks.py`: **Основной скрипт для запуска тестов производительности** всех реализованных алгоритмов.
*   `generator_cases.py`: **Скрипт-генератор тестовых наборов данных** (например, случайных массивов, отсортированных массивов и т.д.).
*   `abstract_sorting_algorithm.py`: **Абстрактный базовый класс для всех сортировок**, определяющий метод `sort()
*   `bubble_sort.py`: Сортировка пузырьком.
*   `bucket_sort.py`: Блочная сортировка.
*   `counting_sort.py`: Сортировка подсчетом.
*   `heap_sort.py`: Пирамидальная сортировка (сортировка кучей).
*   `insertion_sort.py`: Сортировка вставками.
*   `quick_sort.py`: Быстрая сортировка.
*   `radix_sort.py`: Поразрядная сортировка.
*   `selection_sort.py`: Сортировка выбором.

## Функциональность

Основная цель данной лабораторной работы — сравнить асимптотическую сложность и реальную производительность различных алгоритмов сортировки на разных типах входных данных (отсортированных, частично отсортированных, случайных, обратных).

Выводы:

*   Изучить реализацию **Стека**.
*   Разобраться в принципах наследования через абстрактные классы (`abstract_sorting_algorithm.py`).
*   Провести сравнительный анализ **8 различных алгоритмов сортировки**, от простейших квадратичных ($O(n^2)$) до эффективных $O(n \log n)$ и линейных $O(n)$.
*   Автоматизировать процесс тестирования с помощью модуля `kits/`.

Все результаты тестирования и сравнения должны быть представлены в отчете (например, в файле `benchmarks.md`), включая графики зависимости времени выполнения от размера входных данных.
